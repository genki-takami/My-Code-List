# 「直感的」議事録

## iOS版

### 説明
```
本格ワープロソフトなしで直感的に議事録を作成できるアプリです！
作成した議事録はPDFにしたり共有機能でSNSやクリップボードにテキストデータとして送信/保存することもできます！
```

### 技術
```
言語：Swift
IDE：Xcode
ツール：CocoaPods
外部ライブラリ：RealmSwift、SVProgressHUD
```

### 開発ヒストリー
```
所属している柔道部の要望で作成。
オンラインミーティング中にPCのワープロソフトで議事録作成は困難。
スマホで片手でデータ入力する必要がある。
フォーマットがあるとより便利。
既存のアプリはオーバースペック。
PDFも作成できるとヨキ。
議事録リストをフォルダで分けることができると便利。
共有機能のテキストデータでチャットアプリにペーストできる。
フォルダ名が変更できないとレビューにあったので修正。
会議の参加者名簿を登録できると議事録作成時に選ぶだけで済むから便利。
会議の場所を登録できると議事録作成時に選ぶだけで済むから便利。
```
