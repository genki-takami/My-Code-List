# Psyなび for 文化祭&お祭り

## Android版

### 説明
```
「Psyなび for 文化祭&お祭り」は文化祭や学園祭、地域の祭りの案内パンフレットをデジタル化し、
デジタル化された文化祭コンテンツをスマートフォンで閲覧・投稿を可能にするユーザー作成投稿型アプリ
です。
※Android版はコンテンツの閲覧のみ
```

### 技術
```
言語：Kotlin
IDE：Android Studio
外部ライブラリ：Realm for Android、Firebase SDK for Android、Maps SDK for Android
```

### 開発ヒストリー
```
＜開発背景＞
所属していた柔道部が大学の文化祭に参加した際、実行委員会が導入していた文化祭アプリに当部活の模擬
店情報が欠けていた。修正を申請したが、開発会社が休業日であったのとAppleの申請には時間がかかるが
故に、模擬店情報が記載されることなく文化祭が終了。そこから、開発企業に依存しない文化祭実行委員会
がアプリ上の文化祭データをいつでも追加・編集・削除できるアプリが必要であると気づく。また、全国の
文化祭や学園祭のナビゲーションアプリを集積するプラットフォームが無かった事と、散在する個別の文化
祭アプリはその年次のもので完結し持続可能ではなく、学校によっては発注料金も高かった。そのため学生
のためのユーザー作成投稿型文化祭ナビゲーションアプリを開発することを決心。

＜初期＞
文化祭リストタブ・お気に入りリストタブ・お知らせリストタブ・その他タブで構成されたナビゲーション
画面を起点に、ブラウジング機能・マイリスト機能・コメント機能を実装。データストレージのバックエンド
をFirebaseで構築。SDKを利用し、リアルタイムなデータ表示を試みる。ブラウジングに関しては、表示し
た画面で逐次データを受信していたことでやや非効率であった。また、お知らせの表示順序が更新順ではな
かった。

＜中期＞
ブラウジング機能に関して、特定の条件下において事前にリスナーを取得しておくことで瞬間的なデータ表
示が改善した。アプリ内にて複雑かつ時間がかかる処理(削除など)をGCPのCloud Functionで実装するよ
うに修正。投票機能などの新たなUIや機能を追加していくことでアプリをよりリッチに。

＜後期＞
お知らせデータに特殊なタイムスタンプを付属して、更新順に表示されるように改善。
アカウント認証を前提としたデータの追加・編集・削除機能の追加を準備中。
```
