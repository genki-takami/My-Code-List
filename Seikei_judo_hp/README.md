# 成蹊大学体育会柔道部ホームページ

## WEBアプリケーション

### 説明
```
成蹊大学体育会柔道部の公式ホームページ。日々のニュースから記事、選手情報、監督情報、大会結果、
スケジュール、道場までのアクセス、アーカイブ、富じごろうbotといったコンテンツが盛り込まれている。
基本データはFirebase上にあり、サイトはSPAで、開発環境と本番環境は分けてある。
コンソールから、サイト上のデータを更新することができる(ログインが必要)。
```

### 技術
```
言語：HTML5、CSS3、JavaScript(Vue.js)
エディタ：Atom
ツール：npm、Vue CLI、Netlify CLI
ホスト：Netlify
外部ライブラリ：axios、vue-agile、vue-js-modal、vue-light-timeline、vue-router、vue-scrollto、vuex
```

### 開発ヒストリー
```
＜開発背景＞
成蹊大学の柔道部には古いホームページがあったが、アカウントが紛失して編集不可能な状態であった。ま
た、サイトの差別化をするためにサイト作成サービスを利用するのではなく、設計要望に対して細かく作成
できるように一から作成するのが望ましかった。柔道部に所属する友人に作成を頼まれ、入部すると同時に
開発に着手。

＜初期＞
HTMLとCSSだけの静的なサイトで作成。デザインは満足のいくものだったが、動きがない上にデータの更新
がデプロイをするエンジニアである私にしか出来なかった。私が大学を卒業した後のことを考え、プログラ
ミングの知識がない部員でもサイトデータを更新できるように、認証機能付きのコンソールを設置する必要
があった。

＜中期＞
Dialogflowで開発した対話型AIbotの富じごろうをサイト上に実装。柔道部に関することや富じごろう自
身のことを聞いたら、適切なアルゴリズムで返答してくれる。

＜後期＞
Vue.jsでサイトをSPAに修正し、ホスト先をNetlifyに変更。認証やデータストレージのバックエンドを
Firebaseで構築。AxiosとVuexを使い、データの送受信を管理・実装することで、認証を済ませたらコン
ソールからサイト上のデータをテキストベースで更新可能に。さらに、様々なライブラリをしようすること
でユーザビリティが向上。また、これまで本番環境のみで開発していたこともあり、環境変数を使い分ける
ことにより開発環境と本番環境を分けることにした。
```
